About me

Currently, I am a Transaction Management Operations Analyst at Mercado Libre. My functions in this role include data fetching, cleaning, analysis and visualization and developing custom software for automatization of operational processes and handmade reports using Python, SQL, Selenium and Tableau.

In Ixpandit I transformed the financial and operations reporting process from manual, Excel-based operation with weeks delay to real-time automatic process based in SQL, Python (Pandas, Tkinter, Numpy), MySql, JavaScript and Google Sheets. I also automatized reports in PowerBI using Python integration.

Torcuato Di Tella University. I have also worked in several parts of the financial ecosystem including a traditional bank, a broker, the local stock exchange, and fintech companies.

In my free time I like to practice programming doing various projects. My last finished one is a classic alien spaceship shooter clone made using Python Pygame library. You can find it in my GitHub repository.
